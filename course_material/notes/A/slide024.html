ODL Types (cont)

One other structured data type builds tuples   (cf. C's struct). 

Struct Name {
  Type1 FieldName1,
  Type2 FieldName2,
  ...
  Typen FieldNamen,
}

Examples: 

Struct Date {integer day, integer month, integer year}
Struct Name {string family, string given}
Struct Address {integer number, string street, ...}
