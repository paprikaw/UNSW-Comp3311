ODL Model for Bank

Kinds of objects:
  • name, address, date, ...
  • customer, account, branch, transaction, ...
Some we can model via user-defined types: 

Struct Name { string family, string given };
Struct Address {
  integer number;
  string  street;
  string  suburb;
  integer postcode;
};
Enum Activity { deposit, withdrawal };
Enum AcctType { savings, cheque, credit };
