❖ Assignment 3 (cont)

Full IMDB is large (5GB), so we use a cut-down version

  • no TV shows, games, short films, just movies
  • only movies since 1950 and with good ratings
Some IMDB terminology
  • Names = people/animals/CG involved in movies
  • Aliases = releases of movies in different countries
  • Principals = important Names in a movie
  • Acting_roles = what characters an actor plays
  • Crew_roles = what job a crew member does

  • academic information, human resources, accounting

❖ PostgreSQL @ CSE


How it's set up ... (details in Prac 02) ...

  • each student gets a directory /srvr/${USER}/ on Grieg
  • create this directory by running "priv srvr" once on Grieg
  • "pginit" command creates directories and files for PG server
  • including the env script to set your environment to run a PG server
  • after "source env",  "pg 1" starts the server,  "pg 0" stops it
  • on Grieg, PG server runs in  /srvr/${USER}/pgsql/
  • can also run PostgreSQL server on most CSE lab machines (in  /tmp)
